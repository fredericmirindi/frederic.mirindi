Advanced Microeconomics

Consumer Theory

Utility Maximization Problem:

$$\max_{x_1, x_2} U(x_1, x_2) \text{ s.t. } p_1x_1 + p_2x_2 \leq m$$

where \(U(x_1, x_2)\) is the utility function, \(p_1\) and \(p_2\) are prices, and \(m\) is income.

Max Utility Function

Problem Formulation:

$$\max_{x,y} U(x,y) = \max_{x,y} \{\max\{5x,4y\}\} \text{ s.t. } p_x x + p_y y = m$$

Solution Approach:

  1. If \(5x > 4y\), then \(U(x,y) = 5x\). The optimal solution is \(x^* = m/p_x\), \(y^* = 0\), and \(U^* = 5m/p_x\).
  2. If \(5x < 4y\), then \(U(x,y) = 4y\). The optimal solution is \(x^* = 0\), \(y^* = m/p_y\), and \(U^* = 4m/p_y\).
  3. If \(5x = 4y\), any combination of \(x\) and \(y\) satisfying \(p_x x + p_y y = m\) and \(5x = 4y\) is optimal, with \(U^* = 5m/p_x = 4m/p_y\).

Min Utility Function

Problem Formulation:

$$\max_{x,y} U(x,y) = \max_{x,y} \{\min\{5x,4y\}\} \text{ s.t. } p_x x + p_y y = m$$

Solution Approach:

  1. Set \(5x = 4y\) to find the optimal ratio of \(x\) to \(y\): \(x/y = 4/5\).
  2. Substitute this ratio into the budget constraint to solve for \(x\) and \(y\): \begin{align*} p_x x + p_y (5x/4) &= m \\ (4p_x + 5p_y)x/4 &= m \\ x^* &= 4m / (4p_x + 5p_y) \\ y^* &= 5m / (4p_x + 5p_y) \end{align*}
  3. The maximum utility achieved is: $$U^* = \min\{5x^*, 4y^*\} = 5x^* = 4y^* = 20m / (4p_x + 5p_y)$$

Producer Theory

Profit Maximization Problem:

$$\max_{q} \pi(q) = pq - c(q)$$

where \(\pi(q)\) is the profit function, \(p\) is the price, \(q\) is the quantity produced, and \(c(q)\) is the cost function.

Market Equilibrium

Market Clearing Condition:

$$Q_d(p) = Q_s(p)$$

where \(Q_d(p)\) is the demand function and \(Q_s(p)\) is the supply function.

Game Theory

Nash Equilibrium:

$$u_i(s_i^*, s_{-i}^*) \geq u_i(s_i, s_{-i}^*) \text{ for all } s_i \in S_i \text{ and for all } i$$

where \(u_i\) is the utility function of player \(i\), \(s_i\) is the strategy of player \(i\), and \(S_i\) is the strategy set of player \(i\).

Advanced Macroeconomics

Solow Growth Model

Steady State Capital per Worker:

$$k^* = \left(\frac{s}{n + g + \delta}\right)^{\frac{1}{1-\alpha}}$$

where \(k^*\) is the steady state capital per worker, \(s\) is the saving rate, \(n\) is the population growth rate, \(g\) is the technological progress rate, \(\delta\) is the depreciation rate, and \(\alpha\) is the output elasticity of capital.

IS-LM Model

IS Curve:

$$Y = C(Y - T) + I(r) + G$$

where \(Y\) is output, \(C\) is consumption, \(T\) is taxes, \(I\) is investment, \(r\) is the interest rate, and \(G\) is government spending.

LM Curve:

$$\frac{M}{P} = L(r, Y)$$

where \(\frac{M}{P}\) is real money supply, \(L\) is money demand, \(r\) is the interest rate, and \(Y\) is output.

Econometrics

Multiple Regression

Multiple Regression Model:

$$Y_i = \beta_0 + \beta_1X_{1i} + \beta_2X_{2i} + \ldots + \beta_kX_{ki} + \varepsilon_i$$

where \(Y_i\) is the dependent variable, \(X_{1i}, X_{2i}, \ldots, X_{ki}\) are the independent variables, \(\beta_0, \beta_1, \ldots, \beta_k\) are coefficients, and \(\varepsilon_i\) is the error term.

Difference-in-Differences (DiD)

DiD Estimator:

$$\hat{\beta}_{DiD} = (\bar{Y}_{T,1} - \bar{Y}_{T,0}) - (\bar{Y}_{C,1} - \bar{Y}_{C,0})$$

where \(\hat{\beta}_{DiD}\) is the DiD estimator, \(\bar{Y}_{T,1}\) and \(\bar{Y}_{T,0}\) are the average outcomes for the treatment group before and after the intervention, and \(\bar{Y}_{C,1}\) and \(\bar{Y}_{C,0}\) are the average outcomes for the control group before and after the intervention.

Regression Discontinuity (RD)

RD Estimator:

$$\hat{\beta}_{RD} = \lim_{x \downarrow c} \mathbb{E}[Y_i | X_i = x] - \lim_{x \uparrow c} \mathbb{E}[Y_i | X_i = x]$$

where \(\hat{\beta}_{RD}\) is the RD estimator, \(c\) is the cutoff point, \(Y_i\) is the outcome variable, and \(X_i\) is the running variable.

Machine Learning in Economics

Bond Market Predictions

Machine learning techniques, such as auto-regressive integrated moving average (ARIMA), linear regression, and deep recurrent neural network-long short-term memory (DLSTM) models, have been used to predict bond yields and returns in future bond markets. DLSTM models have shown the most accurate predictions over both short and long horizons compared to ARIMA and linear regression.[1]

The ARIMA model can be represented as:

$$ARIMA(p,d,q): (1-\sum_{i=1}^p \phi_i L^i)(1-L)^d X_t = (1+\sum_{i=1}^q \theta_i L^i)\varepsilon_t$$

where \(p\) is the order of the autoregressive term, \(d\) is the degree of differencing, and \(q\) is the order of the moving-average term.

For more information on ARIMA models, see: https://en.wikipedia.org/wiki/Autoregressive_integrated_moving_average

Stock Market Forecasting

Linear regression and random forest models have been employed to predict the future closing prices and probabilities of rise and fall for stocks like Uber. These methods, along with more advanced techniques such as radio frequency technology and neural networks, can be used for stock forecasting.[2]

A simple linear regression model can be expressed as:

$$y = \beta_0 + \beta_1 x + \varepsilon$$

where \(y\) is the dependent variable (e.g., stock price), \(x\) is the independent variable (e.g., time), \(\beta_0\) is the intercept, \(\beta_1\) is the slope, and \(\varepsilon\) is the error term.

For more information on linear regression, see: https://en.wikipedia.org/wiki/Linear_regression

Credit Card Fraud Detection

Machine learning algorithms, including Logistic Regression, Decision Tree, Random Forest, and XG Boost, have been used to detect credit card fraud. The combination of imbalanced classification methods, such as Synthetic Minority Random Oversampling Technique (SMOTE) and Under-sampling, can help balance the dataset and improve the performance of these algorithms.[3]

Logistic regression is a popular method for binary classification problems, such as fraud detection. The logistic function can be written as:

$$P(Y=1|X) = \frac{1}{1+e^{-(\beta_0+\beta_1 X)}}$$

where \(P(Y=1|X)\) is the probability of the positive class (e.g., fraud) given the input features \(X\), and \(\beta_0\) and \(\beta_1\) are the model parameters.

For more information on logistic regression, see: https://en.wikipedia.org/wiki/Logistic_regression

Tax Collection Optimization

Machine learning techniques have been explored to enhance the efficiency of tax debt collection. By using machine learning for taxpayer segmentation, local tax authorities can accurately identify high-risk debtors and optimize revenue collection while minimizing resource waste.[4]

One approach for taxpayer segmentation is k-means clustering, which aims to partition n observations into k clusters, minimizing the within-cluster sum of squares:

$$\min \sum_{i=1}^k \sum_{x \in S_i} ||x - \mu_i||^2$$

where \(S_i\) is the set of points in the i-th cluster, and \(\mu_i\) is the centroid (mean) of the points in \(S_i\).

For more information on k-means clustering, see: https://en.wikipedia.org/wiki/K-means_clustering

Anomaly Detection in Stock Prices

Autoencoder and Light Gradient Boosting Machine have been compared for their performance in predicting anomalies in stock prices. A statistical approach is used to pre-test the data for normality and assign labels. Detecting strange turning trends in stock prices can help investors avoid losses and potentially gain profits.[6]

An autoencoder is a type of neural network that learns to compress and reconstruct input data. It consists of an encoder and a decoder, and the objective is to minimize the reconstruction error:

$$\min ||X - \hat{X}||^2$$

where \(X\) is the input data, and \(\hat{X}\) is the reconstructed data.

For more information on autoencoders, see: https://en.wikipedia.org/wiki/Autoencoder
